### CSS: Медиазапросы

```
Настройки в style.css:
    header li {
      display: inline-block; /* выстроить пункты меню в одну строку*/
    }

    header a {
        text-decoration: none; /* чтобы ссылки не подчеркивались снизу */
    }
    ... (комментарии в style.css)

```
