### HTML: Таблицы
```
Разметка содержит в себе несколько основных элементов:

<table> — область таблицы
<tr> — строка
<thead> — шапка таблицы
<th> — ячейка шапки
<tbody> — тело таблицы
<td> — ячейка


```
