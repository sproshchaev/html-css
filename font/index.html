Удобным способом задать свойства для шрифта является использование правила font, которое позволяет указать: font-style, font-variant, font-weight, font-size / line-height (эти два правила записываются через слэш), font-family. Шесть разных правил внутри одного - это помогает сократить CSS код. Обязательными из них являются font-size и font-family, остальные можно не указывать, пример:

Жирное начертание. Размер шрифта: 24px. Шрифта Verdana. Семейство шрифтов: без засечек.
